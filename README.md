# pbl
🖐️ Hand Gesture Based Browser Control

This project allows controlling a web browser using hand gestures with the help of a webcam.
It uses OpenCV and MediaPipe for real-time hand tracking, NumPy for calculations, and PyAutoGUI / Pynput for simulating keyboard & mouse actions.
Gestures like scroll, tab switch, and navigation are detected and executed automatically.
The project demonstrates Human-Computer Interaction (HCI) with computer vision.
